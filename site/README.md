# 企业管理自测站上线说明

这个文件夹是干净的静态网站上线包，核心文件是 `index.html`。

附带文件：

- `converter.html`：把普通题目/答案文本转换成网站可导入的 JSON。
- `template.json`：手工整理题库时可参考的标准格式。

推荐方式：

1. 打开 https://app.netlify.com/drop
2. 把整个 `site` 文件夹拖进去
3. 等待生成网址
4. 手机直接打开生成的网址即可使用

也可以上传到 Cloudflare Pages、GitHub Pages、Vercel 等静态网站平台。

说明：

- 这是纯静态网页，不需要服务器程序。
- 网站内可以点“导入题库”导入 JSON，也可以点“导出题库”保存当前题库。
- 作答记录保存在浏览器本机的 localStorage 里。
- 手机和电脑的作答记录不会自动同步。
- 上线后不需要这台电脑保持开机。
